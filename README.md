**iMoedas** é o primeiro challenge de código desenvolvido no **Apple Developer Academy - IFCE**. O objetivo do projeto foi consolidar os fundamentos do desenvolvimento iOS, aplicando boas práticas de arquitetura, interface com o usuário e integração com dados persistidos.

## 📱 Sobre o App

O aplicativo permite ter manter um controle financeiro de ganhos e gastos de forma rápida e intuitiva. Durante o desenvolvimento, foram explorados conceitos como:

- Uso de `SwiftUI` para construção da interface
- Gerenciamento de estado com `@State`
- Persistência local de alterações com `@SwiftData` 

> *O repositório foi criado como parte do processo de aprendizado, com foco em versionamento, colaboração e entrega de código funcional.*

## ✨ Funcionalidades

- Adição, remoção, edição e exclusão de movimentações financeiras
- Saldo total, de entradas e de saídas
- Interface limpa e responsiva

## 🛠 Tecnologias Utilizadas

- **Swift** – Linguagem principal
- **SwiftUI** – Framework de interface
- **SwiftData** – Persistência de dados locais

## 🚀 Como Executar o Projeto

1. Clone o repositório:
   ```
   bash
   git clone https://github.com/ulissesbon/iMoedas.git
   
  ``` 

```

2. Abra o arquivo `iMoedas.xcodeproj` no Xcode.
3. Selecione um simulador ou dispositivo físico.
4. Pressione `Cmd + R` para executar o projeto.

> **Nota:** O projeto requer Xcode 15 ou superior e iOS 17+ como alvo de deployment.
